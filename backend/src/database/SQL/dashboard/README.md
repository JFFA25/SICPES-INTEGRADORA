## Métricas Clave

- Usuarios registrados y activos  
- Reservaciones por estado  
- Ingresos mensuales  
- Pagos (pendientes, verificados, rechazados)  
- Ocupación de habitaciones  
- Deudores activos  

## Uso

1. Abrir `SICPES.nbi` en Navicat  
2. Conectar a MySQL  
3. Ejecutar consultas  
4. Visualizar o exportar resultados
