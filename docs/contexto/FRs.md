# Requerimientos Funcionales (FRs)

| ID | Nombre | Descripción |
|---|---|---|
| FR-01 | Registro de Usuarios | El sistema debe permitir nuevos registros de usuarios definiendo su rol (alumno, empleado). |
| FR-02 | Autenticación Segura | El sistema debe permitir iniciar sesión y validar acceso mediante tokens de seguridad (JWT). |
| FR-03 | Control de Habitaciones | El administrador debe poder visualizar, registrar, actualizar estado y eliminar habitaciones. |
| FR-04 | Gestión de Reservaciones | Los estudiantes deben poder solicitar reservaciones de habitaciones disponibles. |
| FR-05 | Aprobación de Reservaciones | El administrador podrá cambiar el estatus de las reservaciones (Ej. Aprobada, Rechazada). |
| FR-06 | Gestión y Validación de Pagos | El sistema debe de registrar y administrar los comprobantes de pago hechos por los estudiantes. |
| FR-07 | Control de Roles | La interfaz gráfica debe adaptarse permitiendo operaciones de mantenimiento solo a los roles administrativos. |
