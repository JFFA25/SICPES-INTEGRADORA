# Requerimientos No Funcionales (NFRs)

| ID | Nombre | Descripción |
|---|---|---|
| NFR-01 | Rendimiento | Las respuestas de la API deben retornar en menos de 2 segundos en condiciones de tráfico habituales. |
| NFR-02 | Seguridad | Las contraseñas en bases de datos deben almacenarse con un algoritmo de encriptamiento robusto y no en texto plano (e.g. Bcrypt). |
| NFR-03 | Usabilidad | La interfaz de usuario desarrollada en React y Tailwind CSS debe ser responsiva y adaptarse a dispositivos móviles, tablets y escritorio. |
| NFR-04 | Arquitectura | El sistema debe estructurarse utilizando el modelo Cliente-Servidor separando Frontend (React) y Backend Node.js (Express). |
| NFR-05 | Escalabilidad | La base de datos debe ser capaz de soportar el registro de cientos de peticiones de estudiantes durante el periodo inicial de cada ciclo escolar de forma concurrente. |
