# Reglas de Negocio (BRs)

| ID | Nombre | Descripción |
|---|---|---|
| BR-01 | Restricción de Solicitudes | Un usuario (estudiante) no puede tener más de una reservación activa o en estado pendiente al mismo tiempo. |
| BR-02 | Visualización de Habitaciones | Solo se ofertarán y mostrarán las habitaciones cuyo estado sea "disponible". Habitaciones en mantenimiento no aplican. |
| BR-03 | Política de Rechazos | Al ser rechazada una reservación, la habitación ligada debe regresar a ser elegible para otros estudiantes. |
| BR-04 | Asignación de Permisos | Solo aquellos usuarios designados explícitamente con el rol 'admin' están facultados para modificar, aprobar o anular pagos y reservaciones. |
